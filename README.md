HR Analytics Dashboard | Power BI
📊 Project Overview

The HR Analytics Dashboard is an interactive Power BI project designed to analyze employee attrition and workforce demographics. It helps HR professionals identify key factors influencing employee turnover through dynamic visualizations and KPIs.

This dashboard provides insights into employee count, attrition rate, salary distribution, age groups, education background, years at the company, job roles, and gender distribution.
